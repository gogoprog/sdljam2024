#SDLJam 2024

Entry for the SDL Discord - 2024 Summer Game Jam!

It has only one dependency : `SDL2` and `SDL_mixer` and `box2d`. Did I say one?

## Features

  * Terrain rendering
  * Path finding
  * Physic collisions
  * Units controlling
  * Structures building
  * RTS gameplay
